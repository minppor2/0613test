Vibe Coding 웹페이지 배포 가이드

만들어진 정적 웹페이지(HTML, CSS)를 무료로 전 세계에 배포하는 방법을 소개합니다. 대표적으로 GitHub Pages를 사용하는 과정을 안내합니다.

1. GitHub 저장소 생성 및 푸시

먼저, 로컬에서 작업한 코드를 GitHub 저장소에 업로드(Push)합니다. 방금 진행하신 커밋과 푸시(Push)가 이 과정에 해당합니다.

2. GitHub Pages 활성화

  • GitHub 저장소(Repository) 페이지로 이동합니다.
  • 상단의 Settings 탭을 클릭합니다.
  • 좌측 사이드바 메뉴에서 Pages를 선택합니다.
  • "Build and deployment" 섹션의 Source를 Deploy from a branch로 설정합니다.
  • Branch를 main (또는 master)으로 선택하고 / (root) 폴더를 지정한 후 Save 버튼을 누릅니다.

3. 배포 확인

설정을 저장하면 GitHub Actions가 자동으로 배포를 진행합니다. 약 1~2분 후 상단에 배포 완료된 접속 주소가 표시됩니다.
(예: https://minppor2.github.io/0613test/). 해당 링크를 클릭하면 인터넷 상에서 페이지를 확인할 수 있습니다.
